========================================
PASSO 1 - CRIAR BOT TELEGRAM
========================================

1. Telegram, busca @BotFather
2. Manda: /newbot
3. Nome: ex "Assistente do Jonas"
4. Username (termina em bot): ex jonas_assistente_bot
5. Copia o TOKEN retornado

Pegar ID Telegram do dono:
1. Busca @userinfobot, manda qualquer msg
2. Copia o ID numerico (ex 1672601444)


========================================
PASSO 2 - INSTALAR DEPENDENCIAS (1 COMANDO)
========================================

Linux (root):
   curl -fsSL https://raw.githubusercontent.com/denderson2013-bot/agente-claude-telegram-setup-alunos-denderson/main/bootstrap.sh | bash

macOS (usuario normal):
   curl -fsSL https://raw.githubusercontent.com/denderson2013-bot/agente-claude-telegram-setup-alunos-denderson/main/bootstrap.sh | bash

Instala: Node 22, Python 3, ffmpeg, Claude Code CLI 2.1.118 (pinado),
tmux, PostgreSQL 16 + pgvector, Caddy, pm2 globalmente.

Tempo: 3-5 min.


========================================
PASSO 3 - LOGAR NA CONTA CLAUDE
========================================

   claude auth login --claudeai

Aluno copia o link, abre no navegador, loga (Pro ou Max),
autoriza, copia o codigo, cola no terminal.


========================================
PASSO 4 - INICIAR O CLAUDE
========================================

Linux:
   cd /root && claude --dangerously-skip-permissions

Mac:
   cd ~ && claude --dangerously-skip-permissions


========================================
PASSO 5 - DEIXAR O CLAUDE FAZER O RESTO
========================================

Dentro do Claude, cola:

   Leia o arquivo SETUP-AGENTE.md e execute todos os passos.
   Me faca perguntas quando precisar de informacao minha.

========================================
COMANDOS DO DIA A DIA
========================================

Status geral:
   systemctl status AGENTE
   systemctl status AGENTE-bot
   pm2 status

Logs ao vivo:
   tail -f /opt/AGENTE/logs/agent.log
   tail -f /opt/AGENTE-bot/logs/bot.log
   pm2 logs agent-manager

Tela do Claude:
   su - AGENTE -c "tmux attach -t AGENTE"
   (sair sem fechar: Ctrl+B depois D)

Restart:
   systemctl restart AGENTE
   systemctl restart AGENTE-bot
   pm2 restart agent-manager

Editar personalidade:
   nano /opt/AGENTE/CLAUDE.md && systemctl restart AGENTE

Editar subagente:
   nano /opt/AGENTE/.claude/agents/paulo-dev.md
   (sem restart)

Ver memoria:
   sudo -u AGENTE psql -d AGENTE_memory \
     -c "SELECT COUNT(*) FROM conversation_history"

Ver leads prospect:
   sudo -u AGENTE psql -d AGENTE_memory \
     -c "SELECT * FROM prospect_leads ORDER BY created_at DESC LIMIT 10"

Ver job queue:
   sudo -u AGENTE psql -d AGENTE_memory \
     -c "SELECT id,type,status,attempts FROM job_queue WHERE status != 'done' LIMIT 20"

========================================
PROBLEMAS COMUNS
========================================

"Mensagens duplicadas no Telegram"
   Plugin oficial nao deve estar ativo. Confere:
   cat /home/AGENTE/.claude/settings.json | grep enabledPlugins
   Se tiver telegram, REMOVE e reinicia.

"https://AGENTE.dominio.com da 502"
   Tunnel caiu. tunnel-watchdog.sh deveria recriar em 5 min.
   Manual: pm2 restart agent-manager && systemctl reload caddy

"Naia Rita nao envia DM Insta"
   Tandem aberto no Mac do dono?
   TANDEM_TOKEN valido no .env?
   Limite diario nao estourou? (50-100 DMs/dia)

"Job queue parado"
   crontab -u AGENTE -l | grep process-job-queue
   Roda manual: sudo -u AGENTE python3 /opt/AGENTE/cron-scripts/process-job-queue.py

"Audio nao transcreve / nao sai"
   Confere chaves no .env do bot:
   cat /opt/AGENTE-bot/.env | grep -E 'OPENAI|ELEVEN'

"Agente esquece tudo"
   Confere consolidate-conversations:
   crontab -u AGENTE -l | grep consolidate

"VPS reboot e nao voltou"
   systemctl is-enabled AGENTE AGENTE-bot
   pm2 startup (re-gera comando, executa)
